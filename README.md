# Timer Challenge Game

A simple React project built while learning React fundamentals.

## About

This project is a small timer-based game where the user tries to stop a countdown timer as close as possible to a target time. Different difficulty levels change the target duration.

## Features

- Player name input
- Multiple timer challenges with increasing difficulty
- Start/Stop timer functionality
- Score calculation based on accuracy
- Result modal showing win/loss outcome

## Tech Stack

- React
- JavaScript (ES6+)
- CSS

## Purpose

Built as a learning exercise to practice:

- React hooks (`useState`, `useRef`, `useImperativeHandle`)
- Component composition
- Portals
- State management
